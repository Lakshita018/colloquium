RULE BOOK

RULE NO 1

Web-vida is a dynamic website development competition with a time duration of 2 hours.
The event is scheduled for February 17th, 2025, in UG Activity Spaces 1 and 2, at 5:30 PM.
The event consists of two rounds. Initially, each team will have 500 points.

RULE NO 2

Round 1
It will be a trivia round on web development-related topics. A QR code will be displayed initially, redirecting participants to a website containing questions. After completing the trivia round, a leaderboard will be displayed showcasing the points earned.

RULE NO 3

Round 2
Teams will then participate to bid their favorable website domain for website building using the points earned in the previous round. Following, a QR code will be displayed, redirecting to a GitHub repository containing a basic code template.

RULE NO 4

Teams will be challenged with commands that require them to implement specific features into their website (if completed, extra points will be given). The Website building round will then continue for 45 minutes, after which the teams are required to submit their source file.
Teams will be judged on creativity, uniqueness and functionality of the website.

RULE NO 5

All participants must bring their laptops fully charged with a stable internet connection.
The Organizing Committee reserves the right to disqualify teams found using unfair means, with the decision being final.
